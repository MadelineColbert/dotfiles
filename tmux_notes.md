control + space is the leader key

## Panes
Leader key then arrow keys changes active
Leader key plus x deletes a pane.
Leader key plus ! creates a new window from pane
Leader key plus \ and - creates new panes horizontally and vertically
Holding leader and an arrow key can shrink/grow the pane

## Windows:
Using leader + number switches to that window
Leader + c creates a new window
Leader + & closes current window
Leader + , renames a new window
Leader + w lists windows
Leader + p moves one window back
Leader + n moves a window forward

## Sessions:
Leader + s lists all Sessions
Leader + $ renames Sessions
Leader + d detaches from the session

## Commands:
tmux a -t "session name" - attaches to session
tmux new -s "session name" - creates a new session with session name
